# Stock Phyto 1.6 — Parcellaire PAC 2026

- Import intégré des 65 parcelles du dossier PAC 2026 (117,43 ha).
- Nouvel onglet **Parcellaire** pour renseigner et modifier le nom habituel de chaque parcelle.
- Dans un traitement, sélection directe d'une parcelle du parcellaire.
- Surface entière, 1/2, 1/3, 2/3 ou surface personnalisée.
- Synchronisation des noms et du parcellaire entre téléphone et ordinateur via Supabase.

Avant le premier déploiement, exécuter le fichier `parcels-supabase.sql` dans l'éditeur SQL Supabase.

# Stock Phyto v1.7 – Carte du parcellaire

Cette version ajoute une carte interactive à l'onglet Parcellaire :
- contours exacts des 65 parcelles PAC 2026 ;
- fond de carte OpenStreetMap ;
- recherche et filtrage ;
- fiche de parcelle au toucher ;
- ajout et modification du nom habituel ;
- bouton de géolocalisation ;
- bascule Carte / Liste ;
- ergonomie adaptée au téléphone.

La table Supabase `parcels` créée pour la v1.6 reste valable : aucun nouveau script SQL n'est nécessaire.

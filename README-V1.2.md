# Stock Phyto v1.2

## Nouveautés
- valeur totale du stock sur le tableau de bord ;
- prix d'achat par unité de stock ;
- fournisseur, numéro de lot et date d'achat ;
- filtres Tous / Stock faible / Ruptures / Avec prix ;
- recherche par nom, fournisseur, lot, AMM ou matière active ;
- export Excel enrichi avec Synthèse, Stock détaillé, Mouvements et Préparations ;
- correction de la configuration Browserslist pour la génération PWA.

## Installation
```bash
npm install
npm run dev
```

## Vérification production
```bash
npm run build
```

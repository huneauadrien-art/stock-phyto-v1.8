# Stock Phyto v1.5.2

- Le litrage/ha démarre vide (valeur 0) pour chaque nouveau traitement et chaque nouveau plein.
- La quantité de chaque produit est calculée automatiquement : dose/ha × hectares théoriques.
- Les quantités calculées restent modifiables pour corriger la quantité réellement versée.
- Les champs décimaux acceptent la virgule et le point, notamment les doses/ha, hectares théoriques, quantités et surfaces de parcelles.
- Le calcul d'eau reste automatique : litrage/ha × hectares théoriques.

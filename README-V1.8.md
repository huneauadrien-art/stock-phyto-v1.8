# Stock Phyto v1.8.0 — Sélection parcellaire

Nouveautés :
- sélection de plusieurs parcelles directement sur la carte ;
- parcelles sélectionnées affichées en vert ;
- compteur de parcelles et total automatique des hectares ;
- sélection également disponible dans la vue Liste ;
- bouton « Préparer le traitement » ;
- transfert automatique des noms, de la culture et de la surface vers la préparation ;
- affichage des parcelles choisies dans la préparation ;
- volume d'eau par hectare initialisé à 0.

Le projet a été validé avec `npm run build`.

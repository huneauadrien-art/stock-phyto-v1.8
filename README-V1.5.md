# Stock Phyto v1.5.0 — Traitements simplifiés

## Nouveautés
- Fusion de Préparation et Registre dans un seul onglet **Traitements**.
- Création d'un traitement avec le premier plein, l'eau, les doses/ha et les quantités réellement chargées en une seule saisie.
- Ajout des parcelles après le passage, au fur et à mesure.
- Modification complète d'un traitement, même terminé : date, produits, doses, pleins, quantités et parcelles.
- Recalcul automatique du stock lors d'une correction.
- Suppression d'un traitement avec remise des produits en stock.
- Cartes du tableau de bord cliquables : Produits, Stock faible, Ruptures et Valeur ouvrent directement la bonne liste filtrée.
- Les anciennes adresses /preparation et /registre redirigent vers /traitements.

## Lancement
```bash
npm install
npm run dev
```
